- 无脑一键更新, 适用于新装系统
    ```
    apt install git -y && 
    git clone https://github.com/kevinshane/unlock.git && 
    chmod -R +x /root/unlock/ && 
    /root/unlock/step1-update.sh
    ```

- 无脑一键解锁
  ```
  /root/unlock/step2-unlock.sh
  ```

- 无脑一键切分
  ```
  /root/unlock/next-step.sh
  ```

---

- 也可以无脑一键直通，适用于新装系统
  ```
  /root/unlock/passthr.sh
  ```
